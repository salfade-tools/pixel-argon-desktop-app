# PixelArgon Roadmap

## v1.0 (Current)
- [x] Open PNG/JPEG/WebP images
- [x] Canvas viewer with zoom/pan
- [x] Crop tool with aspect ratio presets and custom size
- [x] Basic edits: grayscale, brightness, contrast, rotate, flip
- [x] Pixelate brush for redaction with undo/redo
- [x] Rust export pipeline (accurate processing)
- [x] Recent files (last 10)
- [x] Keyboard shortcuts
- [x] Background removal (simple chroma key, Beta)
- [x] Auto-updates via GitHub Releases (macOS Apple Silicon)

## v1.1 Ideas
- [ ] Batch processing: apply same crop+edits to multiple images
- [ ] Preset saving for custom A×B sizes

- [ ] Better background removal (ML-based using ONNX/U²-Net)
- [ ] Non-destructive layer stack UI
- [ ] Crop guides (rule of thirds, golden ratio)
- [ ] Image comparison (before/after split view)
